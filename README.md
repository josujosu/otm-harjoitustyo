### Testaus

Testit voi suorittaa komennolla

    mvn test

Testikattavuusraportin voi luoda komennolla

    mvn jacoco:report

Testikattavuusraporttia voi tarkastella avaamalla tiedoston *target/site/jacoco/index.html* selaimella
